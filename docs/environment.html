Environment variables

Prefix MICROCORE_ can be changed by command line parameter -DMICROCORE=newValue

Any environment variable can be set either as usual environment variable as follows:

SET MICROCORE_PATH=/dvv/var

SET MICROCORE_DVNET_LOG=true

or in the command line as follows (which has priority over environment):

microcore   -DMICROCORE_PATH=/dvv/var   -DMICROCORE_DVNET_LOG=true

or in microcore.properties file, which has the highest priority. In microcore.properties you can check #ifndef, #ifdef before setting any variable

dvparser

MICROCORE_PATH a path where microcore.properties is sought if not found in the current directory

dvnet

MICROCORE_DVNET_LOG if set to "true", it logs all net request parameters

dvcrypt

MICROCORE_DVCRYPT_ENCODING_PATTERN 32-character word of Latin letters and digits used in additional non-secure encoding/decoding

MICROCORE_DVCRYPT_KEY_FOLDER a path to the folder where public/private keys are stored, by default "/etc/secrets"
