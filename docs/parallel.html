Parallel Processing

Parallel processing parameters are described at the root level in the config and they are general for the whole microservice.

interval_time_unit is a float number, which describes a time unit for all parallel processes. It is in seconds. For example, 0.5 is a half second. The default is 1 second. The minimum value is 0.01. 

    Example:
    "parallel_processing": {
       "interval_time_unit": 0.5
    }
