dynamic

Dynamically provides properties and actions and immediately enforces them
It has the parameters as follows:
{
    "production" map of {string properties} to be set in
       cloud
    "development" map of {string properties} to be set for
       local development
    "properties" common map of {string properties}
    "json" map  of {json properties}
    "actions" array of {action descriptions}
}
Example:
    DYNAMIC_ACTION_1=dynamic:{"json":{"A1":{"A2":5}}, "actions":[{"name":"A"}]}
