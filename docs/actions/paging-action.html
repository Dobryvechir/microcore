paging

Paging command has the following parameters:
{
     "source" {json source}
     "result" the name of the variable where to store the response
     "prefix" string, the prefix to store the paging data, defaults to "PAGING"
     "pageSize" int, the page size in the number of items per page, defaults to 3
     "currentPage" int, one-based number of the current page, defaults to 1
}
Example:
PAGING_CARDS_1=paging:{"source":{"var":"PAGING_SOURCE"},"prefix":"PAGE","result":"PAGING_RESULT"}
